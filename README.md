# scipm.tictactoe

scipm.tictactoe is a [scipm](https://github.com/aminassian/scipm) package

source : http://lua-users.org/wiki/SciteTicTacToe

## install

```
cd myScipmProject
npm install scipm.extman --save
scipm install scipm.tictactoe --save
scipm build
[restart SciTE]
```

## actions

See tool menu TicTacToe
